# 知栖更新日志

本文件面向安装包使用者，记录每个发布版本的新增、优化和修复。每次发布前同步更新版本号、发布日期和本版本内容；更细的技术记录见 [docs/updates.md](docs/updates.md)。

## 1.1.1 - 2026-07-13

### 新增

- 本机 MCP 现在可由已授权的 AI 客户端创建子页面，并同步显示在父页面正文和侧栏结构树中。
- 白板开放给本机 MCP 读取和增量编辑：可创建流程图、架构图、连线、画笔笔迹与局部擦除。
- 设置中心左下角显示当前应用版本，便于确认已安装版本。

### 优化

- 白板连线支持上下或左右方向、锚点、直线或曲线、颜色和线宽；节点可使用不同形状、配色与层级。
- MCP 写入与本地保存的并发处理更稳妥，避免旧界面快照覆盖刚由 AI 新建的内容。
- 设置中心版式调整：版本号固定在左下角，并移除空白页面中多余的右侧滚动条。

### 修复

- 修复通过斜杠菜单将空白块转换为子页面时，创建空引用导致页面无法打开、侧栏不显示的问题。
- 修复已启用的本机 MCP 服务在设置中心可能显示为未启用的问题。
