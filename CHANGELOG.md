# 知栖更新日志

本文件面向安装包使用者，记录每个发布版本的新增、优化和修复。每次发布前同步更新版本号、发布日期和本版本内容；更细的技术记录见 [docs/updates.md](docs/updates.md)。

## 1.2.3 - 2026-07-15

### 修复

- 修复 Windows 拖入 Word、PDF 或其他附件时，应用可能立即退出的问题。
- 文件复制改用堆内存缓冲，避免主线程栈溢出；左侧和正文区域的导入规则保持不变。

## 1.2.2 - 2026-07-15

### 优化

- Windows 拖入文件时，左侧页面树与右侧正文会分别显示对应的导入说明和高亮状态。

### 修复

- 修复 Windows 安装版拖入 Word、PDF 或附件时可能触发应用异常退出的问题。
- 左侧拖入文档会创建顶级页面，右侧拖入文档会创建当前页面的子页面；附件分别进入收件箱或追加到当前页面。

## 1.2.1 - 2026-07-15

### 修复

- 修复 Windows 打包版拖入文件时，全局收件箱遮罩错误接管正文或侧栏目标、并可能导致界面异常退出的问题。

## 1.2.0 - 2026-07-14

### 新增

- 支持将带文字层的 PDF 本地导入为可编辑页面，并保留原始 PDF 附件。
- 支持将 DOCX、PDF、Markdown、TXT 拖入页面树或当前页面；文档可自动创建顶层页面或子页面。

### 优化

- 普通附件拖入当前页面时，会按顺序追加到正文末尾；拖入页面树空白处时进入收件箱。
- 普通 JSON 文件仍可作为附件保存；知栖页面包和完整备份会要求从专用导入入口处理。

### 修复

- 改善 PDF 被拆分文本的词间空格还原，并避免后续文件导入失败影响此前成功页面的打开。

## 1.1.1 - 2026-07-13

### 新增

- 本机 MCP 现在可由已授权的 AI 客户端创建子页面，并同步显示在父页面正文和侧栏结构树中。
- 白板开放给本机 MCP 读取和增量编辑：可创建流程图、架构图、连线、画笔笔迹与局部擦除。
- 设置中心左下角显示当前应用版本，便于确认已安装版本。

### 优化

- 白板连线支持上下或左右方向、锚点、直线或曲线、颜色和线宽；节点可使用不同形状、配色与层级。
- MCP 写入与本地保存的并发处理更稳妥，避免旧界面快照覆盖刚由 AI 新建的内容。
- 设置中心版式调整：版本号固定在左下角，并移除空白页面中多余的右侧滚动条。

### 修复

- 修复通过斜杠菜单将空白块转换为子页面时，创建空引用导致页面无法打开、侧栏不显示的问题。
- 修复已启用的本机 MCP 服务在设置中心可能显示为未启用的问题。
