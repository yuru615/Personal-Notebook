# 知栖首次使用手册与示例工作区设计

## 目标

将默认的“欢迎使用知栖”从简短说明升级为可直接操作的用户手册。手册既说明核心功能，也在正文内提供可编辑的数据表、白板和思维导图示例，帮助新用户从观察进入实际使用。

## 范围

- 保留一个顶层欢迎页，作为首次进入工作区时的默认页面。
- 页面按以下顺序说明功能：开始记录、页面与块、组织与搜索、媒体与导入、数据表、白板、思维导图、设置与备份。
- 在页面中内联插入三个真实可编辑资源块：任务数据表、白板示例、项目规划导图。
- 白板示例包含便签、文本、基础图形和连接线；导图示例包含中心主题、三个一级分支和可继续编辑的子节点；数据表包含名称、状态、日期和备注示例记录。
- 新安装的空工作区直接生成完整页面与三个示例资源。
- 已有工作区只执行一次迁移：未改动的旧欢迎页升级为完整手册；已被用户改动的欢迎页保持原样，另建一页“知栖使用手册”，不覆盖用户内容。
- 用户删除手册后不再自动恢复。

## 数据与迁移

欢迎页种子从单一页面扩展为“页面 + 白板 + 思维导图 + 数据表”组合。页面中的资源块引用同一次种子创建的资源 ID，所有资源均作为普通工作区数据保存，可随页面继续编辑、导出和删除。

工作区设置保存欢迎页 ID 与手册版本。启动时检查版本：

1. 空工作区：创建当前版本的完整手册和示例资源。
2. 旧系统欢迎页：通过标题和旧版固定内容签名识别，原地升级内容并补齐资源。
3. 内容签名不匹配：视为用户修改过，保留页面，创建独立手册页与资源。
4. 已完成当前版本迁移或用户删除手册：不再创建或修改任何内容。

## 页面内容

### 开始记录

说明收件箱、创建页面、标题与图标，以及如何从空白行开始输入。

### 页面与块

说明 `/` 插入菜单、段落、标题、待办、列表、表格、代码、媒体、子页面、引用块和同步块；说明块手柄可用于插入、转换、复制和删除。

### 组织与搜索

说明页面树的展开折叠、星标置顶、页面属性、双向链接和全局搜索。搜索说明明确覆盖正文、媒体文件名与说明、白板文字、导图节点和数据表记录。

### 媒体与导入

说明拖拽文件到收件箱、从剪贴板粘贴图片、Markdown 导入、页面导出和完整工作区备份恢复。

### 数据表实操

嵌入“任务数据表”块，预置三条任务与状态、日期、备注字段，提示用户可切换表格、看板、日历和甘特图视图。

### 白板实操

嵌入“白板示例”块。画布上以便签、文本、矩形和箭头展示“收集 -> 整理 -> 行动”的最小流程，并在便签文本中说明选择、拖动、连线、缩放和撤销重做。

### 思维导图实操

嵌入“项目规划导图”块。以“项目规划”为中心，包含“目标”“任务”“复盘”分支及子节点，说明双击编辑、Tab 新增子节点、Enter 新增同级节点和拖动调整结构。

### 设置与备份

说明设置中心中的外观、侧边栏、编辑、导入导出和桌面端选项；提醒定期导出完整工作区备份。

## 实现边界

- 复用现有白板、导图和数据表模型，不引入新的资源类型、运行时依赖或专用教学 UI。
- 示例资源只作为欢迎页内联块存在，不额外在侧边栏创建演示分组。
- 不改变已有工作区的用户页面、白板、导图、数据表和设置；仅按上述迁移规则增加或升级系统说明内容。

## 验证

- 种子测试验证完整页面与三种资源均生成，资源块引用正确。
- 工作区启动测试验证空工作区创建、旧系统欢迎页升级、已编辑欢迎页保留并另建手册、删除后不重建。
- 白板、导图和数据表的既有模型测试继续通过。
- 运行前端构建，并在桌面端检查三个内联示例能打开和编辑。
